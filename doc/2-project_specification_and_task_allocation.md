## 작업 범위

- 임베디드 시스템

    - [ ] 광총에서 버튼 또는 터치 입력을 게임으로 COM 포트를 통해 전달

    - [ ] 버튼 또는 터치 입력 시 일정 지연 시간(필요 시) 후 조도센서 값 / 카메라 정보 읽기

    - [ ] 성공 / 실패 판별 후 판정 결과를 게임(시리얼 포트)과 앱(블루투스)으로 전달

- 게임

    - [ ] 기본 게임 동작

    - [ ] COM 포트를 통해 입력 감지 시 화면 전환을 통해 히트 박스에 해당하는 부분을 표시

    - [ ] COM 포트를 통해 판정 결과를 게임에 반영

- 블루투스 앱

    - [ ] 기본 UI 동작

    - [ ] 블루투스를 통해 현재 게임 상태 표시
